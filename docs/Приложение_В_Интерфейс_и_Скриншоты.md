# ПРИЛОЖЕНИЕ В. ГРАФИЧЕСКИЙ ИНТЕРФЕЙС И СКРИНШОТЫ СИСТЕМЫ

В данном приложении приведено подробное описание дизайн-системы, используемых стилистических решений и шаблонов отображения ключевых экранов интерфейса программного комплекса **CYBER_OPS://CORE**. 

В каждом подразделе подготовлены специальные интерактивные блоки-рамки (слоты) для вставки реальных скриншотов запущенного веб-приложения с подробными инструкциями по их подготовке.

---

## В.1. Спецификация дизайн-системы и визуального стиля

Графический интерфейс разработан с применением принципов **Cyberpunk Glassmorphism** (эффект полупрозрачного стекла с размытием заднего плана) для создания атмосферы хакерского терминала будущего.

### Цветовая палитра (Core Palette)
*   **Фон (Background)**: Глубокий темный оттенок `#10141a` с радиальным градиентом.
*   **Основной акцент (Cyan)**: `#00e0ff` (glow-эффект для ключевых элементов, кнопок и рамок).
*   **Вторичный акцент (Fuchsia)**: `#d946ef` (используется для премиум-доступа и Черного Рынка).
*   **Критический статус (Red)**: `#ef4444` (индикаторы системных сбоев, White Hat/Black Hat конфликты).
*   **Сетки и границы (Surface Borders)**: `rgba(0, 224, 255, 0.15)` с размытием `backdrop-filter: blur(12px)`.

### Шрифтовая схема (Typography)
*   **Space Grotesk** — футуристичный шрифт с открытыми формами букв (для заголовков модулей).
*   **Inter** — нейтральный гротеск с высокой читаемостью (для основного текста и логов).
*   **JetBrains Mono** — моноширинный шрифт (для вывода консоли, терминала и логов).
